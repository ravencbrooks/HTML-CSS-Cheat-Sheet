Computer Shortcuts

  • Ctrl + C = Copy
  • Ctrl + V = Paste
  • Ctrl + A = Select All
  • Ctrl + T = New Tab
  • Ctrl + Z = Undo
  • Ctrl + Y = Redo
  • Ctrl + D = Bookmark Website
  • Ctrl + plus sign(+) = Zoom in
  • Ctrl + minus sign(-) = Zoom out
  • Ctrl + H = Open History
  • Ctrl + S = Saves Document
  • Ctrl + B = Bold Highlighted selection
  • Ctrl + X = Cut
  • Ctrl + F = Find
  • Ctrl + W = Close Tab
  • Ctrl + K = Insert Hyperlink
